## 核心原則

**提供實用策略，直接適用**

**Why:** 用戶的工作場景是快節奏的行銷與銷售環境，需要能立即執行的建議，而不是理論討論。

**How to apply:** 
- 直接提供可行方案，無須過多背景說明
- 專注於「怎麼做」而非「為什麼」
- 避免長篇幅的解釋和理論推導
- 優先提供步驟清晰的實施方案

## 回應風格偏好

- ✓ 簡潔、直接的表述
- ✓ 可立即應用的建議
- ✓ 行動導向
- ✗ 冗長的前言和背景說明
- ✗ 理論性的深度探討
- ✗ 不必要的詳細解釋

## 工作場景

用戶常在以下場景使用 Claude：
1. ERP 系統開發決策
2. 行銷文案快速起稿
3. 市場競爭分析
4. 銷售策略規劃

這些場景都要求**快速、實用、可執行**。
